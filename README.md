# WinterProject4
小A后台组寒假训练营4-备忘录项目

展示视频可以在b站观看
https://www.bilibili.com/video/BV1P2NGzMEnm/

项目说明有时间会马上补充!!!

前端部分使用ai完成

主要实现功能
1.MYSQL建表与基于JDBC的增删改查
  与第三次作业区别不大
  这次的作业没有用到事务,如果后续需要增加删除用户功能,那么因为需要一并删除外键连接的备忘录数据,所以会需要加入事务

  值得一提的是,对备忘录的查询使用了分页查询,配合前端的瀑布流布局,体验较为流畅
  
2.注册,登录功能
  参考了黑马点评项目
  登录时,会使用工具生成一个随机token,作为key,以String的形式存储到redis中
  并且将用户的json数据存储在服务器中
  比起使用cookie的登录方式,更加安全

  如果后续要提高安全性,需要对sql数据库中的密码进行md5加密
  
3.头像上传功能
  前端发起post请求
  后端会根据用户的id生成文件名和路径
  然后通过io流读取并输出图片文件到指定位置
